Meta Character Limits

  • Primary Text: 125 chars before "See more" (2200 max)
  • Headline: 27 chars recommended (40 max)
  • Description: 30 chars recommended (30 max)
  • CTA: use Meta's preset list in Ads Manager

Copy Pairings to Test First

V1 Pain + Short PT · H2 "Your diet isn't broken. Your data is." — highest friction match.

V2 Speed + Curiosity PT · H4 "Your camera is now a nutrition label." — pattern interrupt.

V4 Social Proof + Story PT · H1 quote — lowest resistance entry.

A/B Testing Note

Test PT style (Short vs Story) before headline variants. Primary text has the highest impact on cost-per-click for cold audiences. Once a winning PT is found, run headline tests within that variant.
